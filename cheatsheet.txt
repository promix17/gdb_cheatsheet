[Основные команды]
	gdb - запуск отладчика
	gdb path - запуск отладчика вместа с загрузкой программы
	file path - загрузка файла
	run - начало выполнения программы
	attach pid - присоединиться к процессу
	deatach - отсоедениться от процесса
	kill - убить процесс
	shell cmd - выполнить команду ОС
[Настройка среды окружения]
	set args - установить аргументы командной строки
	show args - показать агрументы командной строки
	path dir - установить рабочий каталог
	show paths - показать зарегистрированные пути
	show environment [имя-перем] - показать значение требуемой переменной окружения
	set environment имя-перем [=значение] - установить значение требуемой переменной окружения
	unset environment имя-перем - удалить требуемую переменную окружения
[Точки останова на выполнение]
	[break] или [b] - точка останова
		break function - на функцию
		break +/- offset - на заданное смещение в строках относительно текущего контекста
		break linenumber - на заданную строку
		break filename:linenumber - на заданную строку в файле
		break filename:function - на заданную фанкцию в файле
		break *addr - на заданный адрес
		break - на следующую инструкцию
		break ... if ... - с условием
	tbreak - одноразовая точка останова
	hbreak - аппаратная точка останова
	thbreak - одноразовая аппаратная точка останова
	info breakpoints - информация о всех точках останова
[Точки останова на обращение к памяти]
	watch expression - на изменение
	rwatch expression - на запичь
	awatch expression - на обращение
	info watchpoints - информация о всех точках останова
[Точки перехвата]
	catch throw - на исключение
	catch catch - на перехват исключения
	catch exec - на вызов
	catch fork - на вызов 
	catch vfork - на вызов 
	catch load - на загрузку библиотеки
	catch load name - на загрузку заданной библиотеки
	catch unload - на выгрузку библиотеки
	catch unload name - на выгрузку заданной библиотеки
	tcatch - одноразовый перехват
	info break - информация о всех точках пеерхвата
[Удаление точек останова]
	clear - удаление текущей точки останова
	clear expression - удаление выбранной точки останова
	delete [breakpoints] [range...] - удаление всех точек останова из заданного диапазона
[Управление точками останова]
	disable [breakpoints] [range...] - отключить указанные точки останова
	enable [breakpoints] [range...] - включить указанные точки останова
	enable [breakpoints] once range... - временно включить указанные точки останова
	enable [breakpoints] delete range... - включить указанные точки останова до первого срабатывания, а затем уничтожить
	commands [номер] - задать требуемые команды на конкретную точку останова
		...list...
	end
[Отладка]
	continue - возобновить выполнение программы
	step - выполнить строку с заходом
	step n - выполнить n строк с заходом
	next - выполнить строку без захода
	next n - выполнить n строк без захода
	finish - продолжить работу до выхода из функции
	return - немедленно выйти из функции
	until - пропуск цикла
	until expression - выполнение до указанного выражения
	stepi - ассемблерная команда с заходом
	nexti - ассемблерная команда без захода
	jump - прыжок на адрес
	call - вызов функции
[Стек]
	frame expression - перемещение по кадрам стека
	backtrace - раскрутка стека
	info frame - информация о стеке
[Исходный код и дизассемблирование]
	list - вывод строк
	disassemble - дизассемблирование
	set disassembly-flavor intel - переход в нормальный ассемблер
[Работа с памятью] 
	set write on/off - управление доступом
	show write - показать параметры доступа
[Расширения]
	source path - выполнить командный файл
[Вывод]
	print - вывод на экран
	x - целое в шестнадцатеричном виде
	d - десятичное целое со знаком
	u - десятичное целое без знака
	o - восьмеричное целое
	t - целое в двоичном виде. 
	a - вывести в виде адреса
	c - рассматривать как целое и вывести в виде строковой постоянной.
	f - число с плавающей точкой
[Исследованеи памяти]
	x/nfu addr - исследование памяти
	n - счётчик повторений
	f - формат отображение в формате printf, i - команда
	u - размер единицы измерений
		b - байты
		h - полуслова (два байта)
		w - слова (четыре байта)
		g - длинные слова (восемь байт)
[Автомотическое отображение]
	display/format выражение - автомотическое отображение, например display/3i $pc
	delete display номера... - удалить номера из отображений
	disable display номера... - деактивировать номера из отображений
	enable display номера... - активировать номера из отображений
	display - вывести на экран
	info display - вывести список отображений
[Регистры]
	info registers - обычные регистры
	info all-registers - все регистры
	info registers name - указанный регистр
	info float - регистры с плавующей точкой
	$sp, $pc - спец регистры
